Chrome doesn't allow extensions to override existing shortcuts automatically. You need to set it up manually — it only takes a few seconds.

1
Open chrome://extensions/shortcuts
(click the link or paste it in your address bar)
2
Find Arc Copy URL and click the pencil icon next to the shortcut field.
3
Press ⌘ Shift C (or any shortcut you prefer) and hit OK.
Why is this needed? Chrome reserves ⌘ShiftC for DevTools by default. Setting it manually tells Chrome you want the extension to take priority.
