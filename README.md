# Arc Copy URL - Chrome extension

A small, beautiful moment in browser history when pressing Ctrl + Shift + C did something magical: it copied the URL of your current tab — instantly.

No right-click.
No clicking the address bar.
No selecting.
Just copy. Done.

Then one day… it disappeared.

This extension brings it back.

Bring back the Arc browser’s beloved shortcut to copy the current tab URL —
or set any shortcut you like.

Fast.
Minimal.
Exactly the way it used to feel.

Built in tribute to the Arc era.

Because some tiny details were just perfect and deserve to live on.
